# 159.261-Games-Programming-Assignment-1-Snake-Game
Here is the code repository of 159.261—Games Programming Assignment 1: Snake Game

In this game,you can play the classic snake game by java files.

At the beginning, you wil see the main menu of the game,and you will see the menu table that tell you how to operate and start a game 

You can click play button by using your keyboard to select which mode you want to play,there are two player mode you can play with.

Click single player mode to play as single player mode,and there will have the snake at the center,you can use the arrow keys to control the snake and change the direction that you want to move.

There would be an good apple which can increase your score and make the snake get longer(but won't get longer than 20).

And it will also have 

Player have 5 lives per game and when you hit yourself,crash to the wall(out of boundary),eat the poison apple, you will lose one live,after you lose the whole 5 lives,you will lose the game and record your score and highest score.So try to get a better score!
